# README Contributors Action

这个项目旨在提供一种简单的方式将项目/仓库的所有贡献者汇总在一张表格里并显示在`README`文件中。

至于为什么需要一个贡献者名单？这是一种鼓励，也是一种表示友好和感谢的方式。
